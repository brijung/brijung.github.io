Lists Usage
Unordered List An unordered list uses a list with a bullet without a need to be shown in any specific order. To start this list, you use the tag <ul>
Ordered List An ordered list uses a list with a number to show items in a specific order. To start this list, you use the tag <ol>
Definition List A definition list has items that come in pairs of terms and definitions. Typically used for glossary, vocabulary, and questions, you start this list using the tag <dl>
